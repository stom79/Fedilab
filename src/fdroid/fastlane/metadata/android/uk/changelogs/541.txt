Додано:
- Додати віддалений перегляд трендів Mastodon як закріплювальну часову шкалу
- Додати налаштування для дозволу завантаження нестисненого медіа (за замовчуванням вимкнено)
- Додати опції для приховування індикаторів «конфіденційного контенту» та «опису медіа»
- Додати опцію для зміни форми зображення профілю на сторінці профілю
- Додати дію спеціальних можливостей для перекладу вмісту статусу
- Новий параметр з кількома опціями для зміни способу відображення кнопки цитати

Змінено:
- Перехід на targetSdk 35
- Покращено параметр попереднього перегляду посилання
- Змінено інтерфейс спливаючого вікна boost+quote
- Видалити обліковий запис з «Головна без звуку» під час скасування підписки
- Зробити URI екземпляра клікабельним у діалоговому вікні «Про екземпляр»

Виправлено:
- Відображати заповнювач для нефедеративних емодзі
- Фільтр повідомлень не працює після завантаження віддаленого профілю
- Анімований WebP не відтворюється в медіа-переглядачі
- Аудіо не клікабельне, коли ввімкнено попередній перегляд за розміром
- Крос-постінг: використовувати правильний екземпляр для обмеження символів та емодзі
- Обрізання не застосовується до цитованих постів
- Попередження про контент втрачено під час редагування запланованих постів
- Переклад DeepL за допомогою заголовка авторизації
- «Відкрити оригінальне повідомлення» може відкривати неправильний статус
- Виправлення, пов'язані зі сповіщеннями про котирування (тип не виключався належним чином, вкладка «Котитування» показувала опитування)

- Проблема з плеєром
- Покращення фільтрів
- Деякі збої
